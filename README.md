# 📦 ST-Git-Sync

## ✨ Особенности

* **Двусторонняя синхронизация (Push & Pull):** Отправляйте свежие изменения из Таверны в облако или подтягивайте актуальную базу данных с GitHub на другое устройство.
* **Умная защита от дубликатов:** Расширение не просто скачивает файлы, а сверяет имена персонажей. При импорте старые версии карточек аккуратно удаляются, не создавая клонов и мусора в вашем интерфейсе.
* **Надежное шифрование (Web Crypto API):** Ваш GitHub PAT токен не хранится в открытом виде. Он надежно зашифрован локально, что защищает ваш аккаунт.
* **Безопасный парсинг:** Встроенная валидация файлов отсекает битые JSONL-чаты и системные плейсхолдеры. Никаких файлов `0.json` или `--- Выберите для редактирования ---.json` в вашем репозитории!
* **Нативная интеграция:** Аккуратное, сворачивающееся меню настроек органично вписано в интерфейс SillyTavern. Кнопки блокируются во время работы, а статус процессов выводится прямо в UI без необходимости открывать консоль.
* **Аварийный сброс:** Встроенная кнопка для безопасной очистки локального кэша Git, если что-то пошло не так.

## ❓ FAQ (Частые вопросы)

**Q: Если я отредактировал карточку или лорбук в Таверне, а потом нажал Pull, что будет?** 
**A:** Расширение честно скачает данные с GitHub и перезапишет ваши локальные файлы. **Всегда нажимайте Push**, если внесли локальные изменения и хотите их сохранить, прежде чем стягивать данные из облака! Расширение выдаст предупреждение перед загрузкой, чтобы вы не потеряли прогресс.

**Q: Какой токен нужен для работы?** 
**A:** Вам понадобится GitHub Personal Access Token (PAT) с правами на чтение и запись в репозитории (Read and Write permissions for Contents). 

**Q: Можно ли использовать публичный репозиторий?** 
**A:** Технически да, но **настоятельно не рекомендуется**. Ваши чаты, профили и мастер-статусы станут доступны всему интернету. Используйте только Private репозитории!

## 🛠️ Установка / Обновление

1. Зайдите в **Extensions** -> **Install Extension**
2. Используйте ссылку: [hhttps://github.com/Gromova-Olga/ST-Git-Sync`](https://github.com/Gromova-Olga/ST-Git-Sync)
3. Если расширение уже установлено — нажмите **Update** во вкладке расширений.
4. Создайте пустой приватный репозиторий на GitHub, сгенерируйте токен и вставьте их в настройки расширения.
